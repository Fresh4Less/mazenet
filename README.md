# Quick Start

## Installation
```bash
$ cd server
$ npm install
$ gulp
$ cd ../client
$ npm install
$ npm run-script build
```

## Usage
```bash
$ cd server
$ npm start
# command line arguments go after --
$ npm start -- --logAll
```

See the `server/` and `client/` READMEs for more specific instructions.
